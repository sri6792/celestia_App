##  What I Built in this App

### NASA Data Integration
APOD Explorer: Pulls the "Astronomy Picture of the Day" using NASA REST APIs.
Space Facts: Curated educational content paired with high-resolution imagery.
Media Lab: Implemented logic for downloading and sharing high-res space assets.

###  Interactive Features & Games
Space Mini-Games: Custom-built interactive logic for space-themed gaming.

Star Map Explorer (VIP): A premium interface for tracking celestial bodies.
Space Relaxation Room: A dedicated UI with ambient sounds and motion effects.

Stargazing Guide: Logic to determine optimal sky-viewing times.

### Backend & UI/UX
Authentication: Full user login, signup, and profile editing via Firebase.
Database: Real-time VIP status tracking and user data stored in Firestore.
Animated UI: Smooth neon-themed transitions using the React Native Animated API.
Navigation: Complex navigation flow using Stack and Tab Navigators.


## Tech Stack
Frontend: React Native (Expo)
Backend: Firebase (Auth & Firestore)
APIs: NASA Open APIs
Styling: Custom UI/UX with neon gradients and motion effects
